
# 小鸢算术 (Xiaoyuan Arithmetic) 🦜

一款专为 5-8 岁儿童设计的“多巴胺配色”趣味数学求和游戏。支持中英双语，拥有动态鼓励系统。

## 🌟 特色功能
- **多巴胺配色**：明亮丰富的视觉设计，激发孩子学习兴趣。
- **双语支持**：一键切换中英文。
- **智能难度**：动态生成求和目标。
- **鼓励系统**：在游戏关键时刻提供可爱的提示和鼓励语。

## 🚀 快速部署到 Vercel

1. **GitHub 同步**:
   - 在本地项目目录执行：
     ```bash
     git init
     git add .
     git commit -m "feat: dopamine ui and multi-language support"
     git branch -M main
     git remote add origin [你的 GitHub 仓库地址]
     git push -u origin main
     ```

2. **Vercel 部署**:
   - 登录 [Vercel](https://vercel.com)。
   - 点击 **New Project** 并关联你的 GitHub 仓库。
   - Vercel 会自动识别 **Vite** 框架。
   - 点击 **Deploy**，等待 1 分钟即可访问。

## 🛠️ 本地开发

```bash
npm install
npm run dev
```
